## 🔐 Security Features

### **Military-Grade Encryption**
- **XChaCha20-Poly1305** authenticated encryption
- **Argon2id** memory-hard key derivation
- **BLAKE3** cryptographic hashing
- **Zero-knowledge architecture** - server never sees data

### **User-Controlled Infrastructure**
- **Private Telegram bots** - each user owns their bot
- **Private channels** - complete control over storage
- **Multi-channel redundancy** - automatic failover
- **No shared infrastructure** - maximum privacy

### **Privacy Protection**
- **Local key storage** - keys never leave device
- **No telemetry** or user tracking
- **Encrypted metadata** storage
- **Audit logging** for security events
